C Shells for Routers & Gateways

May 2, 2019 in posts

Today at work, we made these two C Shells for embedded devices that run busybox. One is a bind shell and the other is a reverse shell.

This was tested on a rooted home gateway that was running of an armv7l CPU. I compiled this on Ubuntu 16.04.6 LTS, therefore needing the cross-compilation libraries and tools.

Installing the libraries & tools:

root@ubuntu:~/asm$ sudo apt-get install gcc-arm-linux-gnueabi binutils-arm-linux-gnueabi libncurses5-dev libc6-armel-cross

Compiling:

root@ubuntu:~/asm$ arm-linux-gnueabi-gcc shell.c -g -o shell

If you need some shells that may work in some OpenWRT Devices, then compile and hack away!
